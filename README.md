## What is SaturnCI?

SaturnCI is a continuous integration platform for Ruby on Rails applications.

## Setup

1. Go to `https://app.saturnci.com/`
2. Click "Sign in with GitHub"
3. Click "GitHub Accounts"
4. Click "Add GitHub Account"
5. Follow the instructions on GitHub
6. Click "Repositories"
7. Click "Add Repository"
8. Choose a repository to add
9. In the terminal in your Rails repository's home directory, run `curl -sSL https://raw.githubusercontent.com/saturnci/website/main/setup.sh | bash`
10. Commit and push
