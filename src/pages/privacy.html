Privacy Policy

As a SaturnCI customer, your code is never used for AI training.

SaturnCI keeps a copy of your code only long enough to run your jobs. When a run finishes, the copy of your repository is deleted from the machine that ran it. Docker images built from your code are cached so that future runs start faster, and deleted after 30 days of disuse.

Test results, logs, and failure screenshots are retained so you can view your run history. If you delete a repository, its data is removed from the product immediately (though residual copies can persist for a time in database backups). If you want a complete purge, email jason@saturnci.com.

Privacy policy last updated June 2026.
